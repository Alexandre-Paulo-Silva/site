Tecnologias

HTML

HTML5 é a linguagem de marcação de hipertexto (HTML) mais recente, a linguagem de programação padrão para descrever o conteúdo e a aparência das páginas da web.

CSS

Abreviado como Cascading Style Sheets, trabalhar com CSS é a próxima etapa do desenvolvimento web, pois define como uma página criada com HTML seria apresentada na internet. Por meio de layouts, fontes, cores, estilos.

PHP

O PHP (Hypertext Preprocessor) é conhecido como uma linguagem de script de propósito geral que pode ser usada para desenvolver sites dinâmicos e interativos.

 Javascript

JavaScript é uma linguagem de programação orientada a objetos leve que é usada por vários sites para criar scripts de páginas da web. É uma linguagem de programação interpretada e completa.

Boostrap

Com o Bootstrap, os desenvolvedores web podem se concentrar no trabalho de desenvolvimento, sem se preocupar com o design, e obter um site de boa aparência instalado e funcionando rapidamente.

Git e GitHub

Um desenvolvedor web precisa ser inteligente e ágil em seu trabalho. Quando algo dá errado no processo de desenvolvimento, não se pode dar ao luxo de começar do zero um projeto novamente. É por isso que é essencial aprender o controle de versão como Git e Github.
